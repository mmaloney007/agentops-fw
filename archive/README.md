# Archive

This folder keeps legacy or superseded artifacts to keep the repo top-level focused on current work.

## Contents
- `archive/legacy/`: legacy top-level modules that have newer versions under `agent_stable_slo/`.
- `archive/papers/`: prior paper drafts and working notes.
- `archive/results/`: older result outputs and training logs.
- `archive/reports/`: older report artifacts (e.g., manifests).
- `archive/criteria/`: older criteria and reward weight configs.
- `archive/diagnostics/`: local diagnostic artifacts (profiling samples, etc.).

If something here becomes current again, move it back to the top-level and update docs accordingly.
